# Contributing to Tickr

Thanks for your interest in contributing.

## Getting Started

1. Fork the repo
2. Create a branch
3. Install deps: `npm install`
4. Run app: `npm run tauri dev`

## Development Notes

- Keep UI changes consistent with existing style direction
- Keep command syntax backwards-compatible where possible
- Prefer small, focused pull requests

## Pull Request Checklist

- [ ] Feature works in `npm run tauri dev`
- [ ] No obvious regressions in setup and running windows
- [ ] Docs updated when behavior changes

## Reporting Issues

Please include:

- OS (Windows/macOS)
- App version
- Steps to reproduce
- Screenshot/recording if possible
