# Tickr Installation Guide

This guide helps users install Tickr on Windows and macOS, including common security warnings.

---

## Download

Latest installers:  
https://github.com/nastarkk/Tickr_App/releases/tag/v0.1.0

---

## Windows Installation (`.msi`)

1. Download the latest Windows installer (`.msi`).
2. Double-click the installer.
3. Complete setup steps.

### If Windows SmartScreen warning appears

You may see a warning like "Windows protected your PC" for unsigned builds.

Steps:

1. Click **More info**
2. Click **Run anyway**
3. Continue installation

---

## macOS Installation (`.dmg`)

1. Download the latest macOS installer (`.dmg`).
2. Open the `.dmg`.
3. Drag **Tickr** to **Applications**.
4. Open Tickr from Applications.

### If macOS blocks the app

For unsigned/notarized builds, macOS Gatekeeper may block first launch.

Option A (recommended):

1. In **Applications**, right-click **Tickr**
2. Click **Open**
3. Click **Open** again in the security prompt

Option B:

1. Open **System Settings** -> **Privacy & Security**
2. Scroll to security message for Tickr
3. Click **Open Anyway**

---

## Notes

- First launch warnings are expected for unsigned builds.
- Signed/notarized builds reduce these warnings significantly.
- If app fails to open, restart the system once and try again.
